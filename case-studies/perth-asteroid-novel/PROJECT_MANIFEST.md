# Project Manifest

## Project

Perth Asteroid Novel

## Purpose

A literary novel project managed like a version-controlled creative repository.

## Current Milestone

Milestone 3: Plot Architecture

## Core Files

- `CANON.md` — accepted story truth
- `DECISION_LOG.md` — history of major creative decisions
- `OPEN_QUESTIONS.md` — unresolved decisions
- `STYLE_GUIDE.md` — tone and prose constraints
- `THEMES.md` — governing thematic architecture
- `AGENTS.md` — agent operating rules

## Major Directories

- `CHARACTERS/` — character architecture
- `PLOT/` — timeline, chapter structure, ending branches
- `WORLD/` — Perth, utopia, seasons, class geography
- `SCIENCE_BIBLE/` — asteroid science and hoax plausibility
- `DRAFT/` — manuscript material and fragments
- `BRANCHES/` — non-canon exploratory ideas
- `../../.agents/skills/` — framework-level agent skills

## Canon Rule

Nothing becomes canon unless it is added to `CANON.md` and recorded in `DECISION_LOG.md`.

## Current Development Focus

Review and stress-test the three-strand plot architecture branch:

- `BRANCHES/plot-architecture-v1.md`

Do not merge it into canon until explicitly instructed.
